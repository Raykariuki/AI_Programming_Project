# Description

This project is a simple rule-based chatbot built using Python and it responds to frequently asked questions about Artifitial Intelligence and Africa International University.
This project also includes a simple data analyzer for sales in a shop which has many different items and there's Average, Total revenue and also visualization of the sales in category form.
This project also includes a Machine Learning model that includes data loading, model training, prediction, and evaluation.

# Features

The Chatbot will be able to handle atleast 5 different users intents and will have polite responses.
The data analyzer is able to scan for any missing values, it will create a total sales column,generate total revenue, generate Average sales and create a bar graph to show the different categories.
The Machine Learning model will be able to load data, Evaluate it, Predict an outcome and also train other model.
